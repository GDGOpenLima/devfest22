Acerca de DevFest Latam 2022

DevFest Latam es una conferencia de tecnología online que se realiza anualmente. En esta edición la organización está compuesta por miembros de diferentes comunidades GDG y GDSC de Latam.

2022 será el 11vo año del DevFest el cual marca el comienzo de una segunda década en las comunidades GDG.

Este evento llevará a cabo de manera online del 03 de diciembre.

Habrán excelentes expositores, regalos, la mejor disposición para compartir contigo y lo mejor de todo: ¡COMPLETAMENTE GRATUITO!.

Talk

Survival of the fittest: Saving UI State in a Compose application

Alejandra Stamato

En esta charla analizaremos como guardar UI state en una aplicacion Android, en los distintos escenarios: configuration changes, activity destruction y system initiated process death. Veremos APIs como SavedStateHandle del AAC ViewModel versus rememberSaveable del lado de la Compose UI. Tambien analizaremos implicancias de manejar manualmente los configuration changes en una Compose app. Que haciamos en el View System? Que podemos hacer en Compose? Guarda esta charla y a tu UI state correctamente!
